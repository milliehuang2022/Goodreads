# Goodreads Book Recommendation System

- Processed 1M+ Goodreads book reviews using NLP techniques like TF-IDF and SentenceTransformers (HuggingFace) to
generate sentence embeddings and compute cosine similarity between textual content for large-scale semantic matching.
- Used Streamlit to build a personalized book recommendation system with sentiment analysis to offer diverse relevant reviews.
